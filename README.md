# Study Planner Application

## Competency  
This project demonstrates my ability to use **Java Spring Boot, MySQL, and Electron** to build a working full-stack software solution.

## Context  
This project was completed as part of my **Java II course**. The goal was to create an application that manages tasks while integrating Java concepts with other technologies.

## Action  
I designed and built the project by creating a **Spring Boot backend** to handle task data and a **MySQL database** for storage. I used **Electron (HTML, CSS, JavaScript)** to create a desktop interface where users can add, view, complete, and delete tasks. The frontend communicates with the backend through REST API calls.

## Result  
The final project successfully stores and manages tasks in a database and displays them in a desktop application. It allows users to **add tasks, view them in a table, filter tasks, mark them complete, and delete them**.

## Reflection  
This project helped me improve my skills in **Java, APIs, database integration, and full-stack development**. If I continued working on it, I would improve the **user interface, add editing features, and implement user accounts**.
